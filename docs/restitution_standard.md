# Standard de restitution

Toute sortie humaine doit suivre le meme format :

1. evenement ;
2. analyse ;
3. observation ;
4. confiance ;
5. risques ;
6. limites ;
7. decision ;
8. prochaine action.

Actions autorisees :

- collecter closing ;
- attendre resultat ;
- relancer evidence gate ;
- observation seulement ;
- refuser.

Actions interdites :

- mise reelle ;
- pick automatique ;
- Telegram agressif.

La restitution doit rester claire, mesurable et prudente. Une bonne restitution affiche aussi ce qui manque : CLV, sample, resultats, qualite de source et risques de fuite.
